## State analysis

Conservative finding: **3 of 4 tickets are coverage gaps, not contract gaps.**

- **BON-602 (✅ already correct):** `mkdir(exist_ok=True)` already preserves `.bonfire/` and `agents/`. New tests cover the gap.
- **BON-601 (✅ already correct):** `_get_loader()` already calls `importlib.resources.files("bonfire")`. The substring assertion in the floor test was weak (a hardcoded `Path(__file__).parent` would also satisfy it). New test mocks `importlib.resources.files` and asserts call-site invocation — the only deterministic strengthening since Python 3.12 returns `PosixPath` from resources-files for regular packages, making type-level "not a Path" checks meaningless.

- **BON-600 (✅ already correct):** Mock `_run_scan` raising KeyboardInterrupt → caught at scan.py:54 → "Front Door closed." prints. Test xpasses. The ticket framing was speculative; the actual implementation handles this case. xfail kept per mission directive (RED state); xpass is non-blocking with default `xfail_strict=False`.
